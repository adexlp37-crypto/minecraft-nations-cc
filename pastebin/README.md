# Imported Pastebin programs

Public ComputerCraft programs imported from [AzadipC's Pastebin](https://pastebin.com/u/azadipC).

These files are archived separately and are **not** installed by the main ComputerCraft updater. Several programs require specific peripherals, coordinates, player lists, web services, or other local configuration before use.

## Security

Exposed Groq API keys and Discord webhook URLs were replaced with placeholders during import. Any credentials that were previously published on Pastebin should be revoked and regenerated at the relevant service.
